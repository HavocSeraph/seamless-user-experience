Section 1 — How to Use This Guide 
This handbook is the authoritative testing reference for SkillBarter — a peer-to-peer skill exchange 
platform powered by a SkillCoin token economy. It is structured as a phase-by-phase prompt suite 
designed to be pasted directly into an AI coding assistant (Claude Opus 4.5, Claude Sonnet 4.5, or 
Gemini 2.5 Pro). Each prompt is self-contained, precise, and written to surface bugs, security 
vulnerabilities, and financial logic errors on the first pass. 
WORKFLOW: Execute phases in order. Each phase depends on the previous. Do not skip Phase 2 
(LedgerService) — it is the financial foundation. A bug in LedgerService means real token loss.
